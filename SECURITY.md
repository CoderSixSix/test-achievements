# Security Policy

## Supported Versions

Currently, only the latest version of this project is supported.

## Reporting a Vulnerability

If you discover a security vulnerability, please report it privately.

### How to Report

1. **Do not** create a public issue
2. Send us a private message through GitHub
3. Include details about the vulnerability
4. Provide steps to reproduce if possible

### What to Expect

- We will acknowledge receipt within 48 hours
- We will provide a response within 7 days
- We will work on a fix as soon as possible
- We will credit you for the discovery

### Security Best Practices

- Keep your dependencies updated
- Review pull requests before merging
- Use strong, unique passwords
- Enable two-factor authentication

Thank you for helping keep this project secure! 🔒
